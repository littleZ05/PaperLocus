# Sample Prompts

## 1. Whole-paper reading from a PDF

```text
Use $paperlocus to read this PDF and produce a structured Chinese reading note.
Focus on the paper's position in the literature, not just the method summary.
```

## 2. Method paper vs science-paper classification

```text
Use $paperlocus to decide whether this Nature paper should be read as a method paper or as a scientific evidence-chain paper.
Explain which narrative signals drive the decision.
```

## 3. Reference-aware understanding

```text
Use $paperlocus to explain this paper in relation to the core prior works criticized in the introduction and the main baselines used in the experiments.
```

## 4. Literature-note production

```text
Use $paperlocus to turn this paper into a reusable Markdown note for my literature graph.
Keep the note compact, reference-aware, and suitable for later retrieval.
```

## 5. Mixed-case science venue paper

```text
Use $paperlocus to read this Nature Methods paper.
Tell me whether it should be understood as a domain foundation-model paper or as a scientific discovery paper.
```
